# Machine Learning Projects

This repository contains machine learning projects developed using Python, Pandas, NumPy, Scikit-Learn, Matplotlib, and Seaborn.

## Projects

### 1. Medical Insurance Charges Prediction

#### Objective
Analyze and predict medical insurance charges using demographic and health-related factors.

#### Dataset
- insurance.csv

#### Techniques Used
- Exploratory Data Analysis (EDA)
- Data Cleaning
- Feature Engineering
- Label Encoding
- One-Hot Encoding
- Feature Scaling
- Correlation Analysis

#### Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Jupyter Notebook

#### Key Insights
- Smoking status significantly affects insurance charges.
- Age and BMI have a positive correlation with charges.
- Feature engineering improves model readiness.
